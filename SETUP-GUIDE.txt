POSTCORP WEBSITE QUICK SETUP

1. Extract this ZIP.
2. Open the extracted folder in Visual Studio Code.
3. Open Terminal and run: npm install
4. Rename or copy .env.example to .env.local
5. Create a PostgreSQL database (Neon is suitable).
6. Paste its connection string after DATABASE_URL= in .env.local
7. Replace JWT_SECRET with a long random value.
8. Choose ADMIN_EMAIL and ADMIN_PASSWORD.
9. Run: npm run db:init
10. Run: npm run dev
11. Website: http://localhost:3000
12. Admin: http://localhost:3000/admin

VERCEL BLOB STORAGE
-------------------
1. In Vercel, open the Postcorp project.
2. Open Storage and create a Public Blob store.
3. Connect the Blob store to the project.
4. Vercel adds BLOB_READ_WRITE_TOKEN automatically to the selected environments.
5. For local testing, copy the token into .env.local.
6. Uploaded images are stored online in Vercel Blob under the public/images/ prefix.
7. Uploaded documents are stored online under the public/documents/ prefix.
8. The CMS stores the returned public Blob URL in PostgreSQL.
9. Uploaded files can be deleted or replaced from the administrator dashboard.
