DATABASE UPDATE
Run npm run db:init again after replacing the project. The schema safely adds the products.gallery_urls column with ALTER TABLE IF NOT EXISTS.

UPLOAD NOTE
Local uploads are saved in public/uploads. This works locally and on a traditional Node/VPS deployment. For Vercel production, connect cloud storage such as Vercel Blob or Cloudinary because Vercel server storage is temporary.
